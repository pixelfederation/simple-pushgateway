Simple Pushgateway – Helm Charts

This GitHub Pages site hosts the Helm chart repository for pixelfederation/simple-pushgateway.

Add this repo to Helm

helm repo add simple-pushgateway https://pixelfederation.github.io/simple-pushgateway
helm repo update

Usage

Install the chart with:

helm install my-gateway simple-pushgateway/simple-pushgateway
